# odin-recipes
The Odin Project HTML first project

## Introduction
This project is part of The Odin Project curriculum and serves as my first HTML project. In this assignment, I built a basic recipe website using only HTML. The website consists of a main index page that links to several recipe pages.

Project Goals
Practice writing structured HTML
Learn how to create and link multiple webpages
Improve understanding of basic website layout and organization
Apply CSS to style and visually enhance a webpage
Skills Demonstrated

##I have demonstrated:

Writing clean and semantic HTML
Styling a webpage using CSS
Using Flexbox to control layout and alignment
Centering elements and aligning content properly
Creating responsive and visually structured layouts
Styling components such as cards, buttons, and navigation elements
Creating links between pages using relative paths
Organizing files into a clear project structure
Using Git and GitHub for version control
Features
Homepage with links to recipes
Multiple recipe pages
Styled recipe cards with images and text
Navigation with a home link
Improved layout using Flexbox
Consistent color scheme and typography

##What I Learned through this project, I learned:

How to build and style a multi-page website
How CSS can be used to improve layout and user experience
How to use Flexbox for alignment and positioning
The importance of spacing, sizing, and visual hierarchy
How to design simple UI components like cards
How to organize and structure a frontend project
